# Styles Directory

This directory contains global styles and CSS modules for the Mahe Tech Systems website.

## Files

- `globals.css` - Global styles and Tailwind CSS imports (located in src/app/)
- Custom CSS modules can be added here as needed

## Design System

The website uses a design system based on the following:

### Colors

- Primary Dark: `#1F3A5F`
- Secondary Blue: `#5F8FB4`
- Background: `#F8FAFC`
- Text: `#111827`
- Accent Gradient: Blue to Light Blue

### Typography

- Headings: Satoshi, Inter, or Manrope
- Body: Inter or DM Sans

### Spacing

- Generous white space
- Mobile-first responsive design

## Tailwind Configuration

All design tokens are configured in `tailwind.config.ts` at the project root.

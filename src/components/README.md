# Components Directory

This directory contains all reusable React components for the Mahe Tech Systems website.

## Structure

- `ui/` - Basic UI components (buttons, inputs, cards, etc.)
- `layout/` - Layout components (header, footer, navigation, etc.)
- `sections/` - Page section components (hero, features, testimonials, etc.)
- `forms/` - Form components (contact form, newsletter, etc.)
- `seo/` - SEO-related components (metadata, structured data, etc.)

## Component Guidelines

1. **Naming**: Use PascalCase for component files (e.g., `Button.tsx`)
2. **Props**: Define TypeScript interfaces for all props
3. **Exports**: Use named exports for components
4. **Styling**: Use Tailwind CSS utility classes
5. **Accessibility**: Ensure all components meet WCAG 2.1 AA standards
6. **Performance**: Optimize for Core Web Vitals
